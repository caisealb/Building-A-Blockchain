WHAT IS A BLOCKCHAIN?

Blockchain technology is a way of recording and managing transaction data, most commonly used in cryptocurrencies like Bitcoin.

A blockchain is made up of units called 'blocks'.

Each block is a package of information that contains three key pieces: data, a hash, and the previous block's hash. A hash is a unique combination of letters and numbers — like a digital fingerprint.

The first block of a blockchain is called the genesis block. Because there are no previous blocks, the value of the previous block's hash is 0.

MAKING A CHAIN

So, each new block is connected to the block before it by that previous block's hash. But what happens when you edit the data of one of the blocks in the chain?

If you edit the data in a block, its hash also changes, making the next block invalid. This is because the next block has an outdated value for the previous block's hash that was edited.

This mechanism helps to make blockchain resistant to tampering or hacking. But the real power of blockchain is shown when we share a chain with others.

A NETWORK OF NODES

Blockchain is often compared to a ledger: a chronological record of transactions. Like a ledger, every time data is added (e.g. through a transaction), a new block — or entry — is added to the chain.

But, unlike a ledger, blockchain isn't kept in one place. It's decentralised, spread over many computers in a network. Each computer is called a node. All nodes have access to the chain — but equally, they must agree to changes in the chain.

To add a block, your computer must solve a crytographic puzzle, known as mining. When it finds a solution, it must share it with the chain's network. The network then decides if the solution is valid, and if the new block can be trusted.

This is called proof of work. If a majority approves the change, the new block is added to the end of the chain.
